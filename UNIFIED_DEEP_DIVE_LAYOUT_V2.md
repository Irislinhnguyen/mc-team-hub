# Unified Deep Dive Layout V2 - Simplified A/B/C Tiers

## Design Principles
- **Simple tiers**: A, B, C (no Hero/Solid/Underperformer labels)
- **Grouped sections**: Separate sections for A, B, C, New, Lost
- **Revenue-sorted**: Within each section, sorted by revenue DESC (highest impact first)
- **Clear transitions**: Show warnings when approaching tier boundaries
- **New/Lost grouping**: Show which tier new items are in, which tier lost items were in
